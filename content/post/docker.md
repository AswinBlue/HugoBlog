# Docker
- 리눅스 커널의 cgroups와 namespace에 의해 제공되는 기술
- 가상화 기능의 일종으로, 별도의 OS를 갖지 않아 VM(Virtual Machine) 보다 가볍다.
- 게스트는 호스트 OS와 자원을 공유한다.
- immutable infrastructure : 서비스 운영 환경을 통째로 이미지화 하여 배포하는 형태
